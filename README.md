# 📝 Laravel Task Manager

A simple and elegant task management system built using **Laravel** with image upload support, due date tracking, and task completion status.

---

## 🚀 Features

- ✅ Create, Read, Update, Delete (CRUD) for tasks
- 🖼 Upload and display task images
- 📆 Set task due dates
- 📌 Mark tasks as completed
- 📋 Responsive UI using Bootstrap 5
- 🗃 SQLite or MySQL support
- 🌐 Deployed for live demo

## 🛠 Tech Stack

- **Framework:** Laravel 10+
- **Frontend:** Blade + Bootstrap 5
- **Database:** SQLite / MySQL
- **Deployment:** Render / Replit / Netlify (optional)
- **Version Control:** Git + GitHub
